# AskMosaic ‚Äì AI Interview Prep App

An **AI-powered Interview Preparation App** built with the **MERN stack** (MongoDB, Express, React, Node.js).  
AskMosaic helps users practice interviews by generating role-based technical questions, answers, and explanations with AI, while providing a smooth and responsive UI.

---

## ‚ú® Features

1. **User Authentication** ‚Äì Register and log in securely with JWT-based authentication.
2. **Role-Based Interview Sessions** ‚Äì Generate questions tailored to job role & experience.
3. **AI-Powered Q&A** ‚Äì Get high-quality technical questions and answers using **Google Gemini API**.
4. **Accordion Learning UI** ‚Äì Expandable/collapsible Q&A for a clean study flow.
5. **Dynamic AI Explanations** ‚Äì On-demand breakdowns and detailed answers.
6. **Pinning Important Questions** ‚Äì Mark and save key questions for quick access.
7. **MongoDB Storage** ‚Äì Store and manage interview sessions & questions.
8. **Responsive Frontend** ‚Äì Built with **React + Tailwind CSS** for modern, clean UI.
